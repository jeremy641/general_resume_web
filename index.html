AI運動賦能系統

Unity 遊戲化 健康科技 Stable Diffusion GPT-3.5

發表論文： ICE ICETA 2024 - "Enhancing Exercise Engagement through AI-driven Virtual Training and Game-Based Animation"

研究目標

透過結合AI虛擬教練與遊戲化互動，提升啞鈴抬舉等重複運動的趣味性與參與度。

系統設計

  • • AI虛擬教練：使用Stable Diffusion自訂教練形象（年齡、外貌、性格）
  • • 智能對話：GPT-3.5生成詳細的教練設定，結合ElevenLab的TTS提供即時語音激勵
  • • 遊戲互動：以Unity開發的互動式"乒乓球"遊戲為基礎
  • • 感測整合：啞鈴內建陀螺儀傳感器，透過舉啞鈴動作控制遊戲進程

系統架構流程圖

實驗結果

實驗設計

  • • 參與者：21名大學生
  • • 分組：傳統運動組 vs 虛擬教練互動組
  • • 測量指標：心率、熱量消耗、運動樂趣評分

主要發現

  • • 心率與熱量消耗無顯著差異
  • • 使用虛擬教練組的運動樂趣評分顯著提升
  • • 參與者表現出更高的持續性與愉悅感

主要結論

虛擬教練與動態遊戲畫面有效提升重複性運動的興趣與參與度，雖然對運動量影響不大，但激發了更高的持續性與愉悅感。

房屋出租速度預測

XGBoost 數據分析 機器學習 QGIS Python

專案概述

聚焦於捷運周邊租屋市場，分析影響出租速度的關鍵因素及趨勢，旨在提高市場資訊透明度。專案利用爬蟲技術收集台灣民間租屋資料，以及來自591租屋網資料集、臺北市資料大平臺、政府開放資料平台的資料 ，結合QGIS和XGBoost回歸模型進行分析。

資料分析

  • • 出租所需天數主要集中在0-10天，顯示大多數房屋出租速度較快
  • • 少數樣本出租所需天數超過20天，形成典型的右偏分布
  • • 採用對數轉換技術，有效消除長尾效應
  • • 使數據分布更加接近正態分布，提升模型預測效能

模型應用

技術特色

  • • 使用XGBoost回歸模型進行分析
  • • 透過正則化處理，減少多變量間的共線性影響
  • • 使特徵權重估計更穩定
  • • 結合地理資訊系統(QGIS)進行空間分析

重要發現

  • • 發現「押金月數」等特徵對出租速度有顯著影響
  • • 押金越高，出租速度越慢
  • • 部分變數如捷運站距離的影響力不顯著，出乎意料
  • • 模型的最佳分數和RMSE顯示其在捕捉整體數據趨勢方面表現出色

應用價值

結果為未來租屋市場策略提供有價值參考，有助於房東制定合理的租金和押金策略，提升出租效率。

強化學習 Unity 教學 遊戲AI 機器學習

專案目標

利用Unity ML-Agent平台，研究與教授強化學習技術的應用。幫助學生理解強化學習的原理並運用於遊戲開發，提升遊戲設計的創意與技術深度。

課程設計

  • • 基礎介紹：Unity ML-Agent工具與安裝設定
  • • 理論講解：強化學習基本概念與原理
  • • 實作練習：從簡單到複雜的專案實作
  • • 應用探索：遊戲AI的實際應用場景

實作項目

  • • 球追方塊：設計虛擬代理學習追蹤目標
  • • 3D環境：創建複雜的3D強化學習環境
  • • 乒乓球比賽：模擬乒乓球比賽的AI對戰
  • • 自適應難度：探索動態難度調節機制

聲動跳龍樁 SoundJumpDragon

Unity PHP 互動藝術 文化科技 聲音互動

展出地點：松山機場、桃園國際機場 T1 & T2、台中國際機場、高雄國際機場

支持單位：昇恆昌「逗陣藝起」計畫

專案簡介

《聲動跳龍樁》是一款結合台灣民俗文化與聲音互動技術的互動藝術遊戲，玩家透過「聲音大小」來控制畫面中舞龍角色的跳躍力道，挑戰不同高度與距離的平台，體驗虛擬舞龍的樂趣。

技術架構

  • • Unity前端：2D橫向跳躍遊戲邏輯，聲音控制跳躍機制
  • • PHP後端：資料儲存伺服器，管理遊玩記錄與成績
  • • 聲音感測：麥克風輸入與實時分貝偵測
  • • 硬體整合：配合各機場現場裝置優化

文化特色

  • • 以「跳龍樁」為核心意象，融合數位互動與地方文化
  • • 強化台灣民俗藝術的數位再現
  • • 提升國際文化交流形象
  • • 吸引不同年齡層觀眾參與
